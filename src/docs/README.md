2. **AI_TASK_QUEUE.md** (`ai/`) — Prioritized tasks; pick `not-started` high-priority items.
3. **DOCS_AUDIT_REPORT.md** (`ai/`) — Compliance audit findings; incomplete tasks, doc inaccuracies.
4. **UNIFICATION_STATUS.md** — Verified unification state; what's done, what remains.
5. **DESIGN_SYSTEM.md** — Color tokens, component API, migration patterns.
6. **ALL_FEEDBACK_CLEAN.md** — Raw owner feedback; append new entries here.
7. **UI_COMPONENT_REFERENCE.md** — Detailed component usage reference.
