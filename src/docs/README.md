# RealmsRPG Docs

## For AI Agents (Cursor)

1. **AGENT_GUIDE.md** (`ai/`) — Component locations, key files, hooks, recording progress.
2. **AI_TASK_QUEUE.md** (`ai/`) — Prioritized tasks; pick `not-started` high-priority items.
3. **ARCHITECTURE.md** — Data flow, Firebase structure, enrichment pipeline, hooks/services.
4. **GAME_RULES.md** — Terminology, formulas, display conventions; use for validation, caps, tooltips.
5. **GAME_RULES_AUDIT.md** — Code vs. rulebook mismatches; audit findings and fix priorities.
6. **DOCS_AUDIT_REPORT.md** (`ai/`) — Compliance audit findings; doc accuracy status.
7. **UNIFICATION_STATUS.md** — Verified unification state; what's done, what remains.
8. **DESIGN_SYSTEM.md** — Color tokens, component API, migration patterns.
9. **ALL_FEEDBACK_CLEAN.md** — Raw owner feedback; append new entries here.
10. **UI_COMPONENT_REFERENCE.md** — Detailed component usage; includes component decision tree.

Root `AGENTS.md` and `.cursor/rules/` provide session-level instructions.

## Archived

`archived_docs/` — Historical migration and audit docs. `ai/templates/` — PR/commit templates.
